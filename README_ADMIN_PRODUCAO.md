## O que dá para editar
- Adicionar ponto/cliente.
- Remover ponto/cliente.
- Mudar nome do ponto.
- Mudar endereço/ponto.
- Mudar horário padrão.
- Adicionar motoca.
- Remover motoca.
- Alterar nome.
- Alterar horário específico.
- Alterar ponto específico.
- Marcar como confirmado ou vaga aberta.
- Exportar/importar JSON de backup.
- Resetar para a base original.

## Importante para GitHub Pages
GitHub Pages é estático. As alterações ficam salvas no localStorage do navegador/dispositivo usado no admin.
Para levar as alterações para outro navegador, use Exportar JSON e depois Importar JSON.

## Confirmação flutuante
Ao selecionar um motoca, aparece o botão flutuante “Confirmar meu horário no WhatsApp”.
A mensagem pronta inclui nome, ponto, dia e horário encontrados para o motoca.
